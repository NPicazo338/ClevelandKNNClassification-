# Cleveland Heart Disease Data Classification

This project is a classification problem that uses the KNN algorithm to classify the presence of heart disease using different attributes obtained from various individuals.

The dataset contains 14 attibutes that describe different biomarkers including age, sex and serum cholesterol levels.
